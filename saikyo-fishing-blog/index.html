Tier System

ティア表の見方

全製品をSS〜Cの5段階で客観評価。価格・釣果・汎用性を総合判断します。

SS
LEGEND
全アングラー必携。釣果・コスパ・汎用性すべてが最高峰の神ルアー
S
ELITE
あれば必ず釣れる。特定条件でSSを凌ぐポテンシャルの上位モデル
A
STRONG
実釣で十分な釣果が出る信頼のスタンダード。ボックスに常備すべき一軍
B
SOLID
特定フィールドや条件で活躍。上位モデルへの買い替えを検討してもよい
C
BASIC
入門・練習用途なら可。釣果を求めるなら上位ティアへのアップグレード推奨
